# Smart Group Evaluator - MERN

একটি সম্পূর্ণ গ্রুপ মূল্যায়ন এবং ব্যবস্থাপনা সিস্টেম

## Features

- ✅ গ্রুপ ম্যানেজমেন্ট
- ✅ ছাত্র ট্র্যাকিং
- ✅ টাস্ক ইভ্যালুয়েশন
- ✅ রিয়েল-টাইম র‍্যাঙ্কিং
- ✅ পারফরম্যান্স অ্যানালিটিক্স
- ✅ PWA Support
- ✅ Dark/Light Theme
- ✅ Responsive Design

## Tech Stack

**Frontend:**
- React 19 + Vite
- Redux Toolkit
- TailwindCSS + shadcn/ui
- Chart.js
- React Hook Form

**Backend:**
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- TypeScript

## Quick Start

### Development

```bash
# Install dependencies
cd client && npm install
cd ../server && npm install

# Run development servers
cd client && npm run dev
cd ../server && npm run dev
```

### Production Build

```bash
# Build frontend
cd client && npm run build

# Build backend
cd server && npm run build
```
